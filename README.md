# yelpCamp

### A fully functional web application designed for camping enthusiasts where users can view camps, add new camp , upload photos, write reviews.

> Integrated cluster map and geocoding for enhanced presentation and user interaction.
> Authentication and Authorization using Passportjs for securing login credentials and resources.
> Sessions for maintaining the application state and persistent login.
> Flash messages for temporary alerts.
> Based on MVC architecture for clean and scalable development.
> Mongoose library for CRUD operation on database.
> Dealing with mongoose and express middleware's.

Technologies Used :-
> Nodejs as runtime environment for executing the javascript on server-side.
> MongoDB for data and session storage.
> ExpressJS to build the Web Server and RESTful API's.
> Bootstrap for styling.
> Maptiler for maps and cloudinary for DAM.
